INTEGRAL/ISGRI with Python
==========================

This is a collection of code to handle `INTEGRAL`_/ISGRI data sets in
Python. It was started as part of the effort of improving the handling
of the instumental background.

.. _INTEGRAL: http://www.cosmos.esa.int/web/integral
